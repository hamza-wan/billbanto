# Billbanto

> Split bills effortlessly — shared expense tracking and group payments, simplified.

Billbanto is a web app that makes it easy for groups to manage shared finances: splitting bills, tracking expenses, and settling up without the back-and-forth. Built with a modern React + TypeScript stack and powered by Supabase.

---

## Features

- **Bill splitting** — divide expenses across group members
- **Expense tracking** — log and monitor shared costs in one place
- **Group payments** — manage who owes what and who has settled up

---

## Getting Started

### Prerequisites

- Node.js (v18+)
- A [Netlify](https://netlify.com) account (for serverless functions)

### Installation

```bash
git clone https://github.com/hamza-wan/billbanto.git
cd billbanto
npm install
```

### Run Locally

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
